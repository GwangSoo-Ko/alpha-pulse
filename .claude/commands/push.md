---
description: 변경사항 커밋 + 푸시
argument-hint: "commit message"
---

Stage all changes, commit with the provided message, and push to origin.

1. `git add -A`
2. `git status --short` to show what will be committed
3. `git commit -m "$ARGUMENTS"` (if no argument, generate a descriptive message)
4. `git push`
